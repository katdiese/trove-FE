Thanks for checking out Trove! Great to have you here.

Trove was created as a capstone project for Galvanize FullStack, graduating June 2016. The purpose of the app is to allow farmers' market goers to check out what is happening at farmers' markets near them or by zip code as well as allow vendors to advertise goods and deals, in real time.

Technologies Used:

Front End: Angular.js, Sass, Skeleton framework, socket.io, firebase deployment, Sketch3 (for wireframing)
Github Repo

Back End: PostgreSQL, knex.js, socket.io, AWS S3, farmers market dataset from Data.gov, heroku deployment
